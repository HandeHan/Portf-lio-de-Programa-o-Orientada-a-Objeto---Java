# **Portfólio de Programa o Orientada a Objeto Java (POO)**

### **Introdução**

Este projeto propõe o desenvolvimento de um sistema para que cidadãos registrem queixas e denúncias sobre problemas da cidade (iluminação pública, buracos, limpeza urbana, saneamento, mobilidade etc.). O foco deste estudo de caso é a parte administrativa da aplicação: o painel (dashboard) onde administradores gerenciam categorias de problemas, visualizam denúncias, acompanham status (aberta/fechada), analisam métricas por mês e gerenciam usuários/denunciantes.
